## Features

- Filter by category: Under $200, sleep apnea detection, temperature control, no subscription
- Visual accuracy bars for sleep stages, heart rate, and apnea detection
- Sensor breakdown and key specs per device
- Honest pros and caveats for each device

## Usage

Open the HTML file directly in any browser. No build step or dependencies required.
